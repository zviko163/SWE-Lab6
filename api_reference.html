Base URL

http://localhost:3000/api

Endpoints

1. Get All Devices

GET /devices

Returns a list of all connected devices.

{
  "devices": [
    { "id": 1, "name": "Living Room Light", "status": "on" },
    { "id": 2, "name": "Thermostat", "status": "off" }
  ]
}

2. Add a New Device

POST /devices

Adds a new device.

Request body:

{
  "name": "Bedroom Light",
  "status": "off"
}

3. Update Device

PUT /devices/:id

Updates the status or name of an existing device.

4. Delete Device

DELETE /devices/:id

Deletes a device by its ID.

5. Get System Status

GET /status

Returns current system health and connectivity.

{
  "system": "healthy",
  "connectedDevices": 4
}
